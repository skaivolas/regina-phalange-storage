ABBYY FineReader 15.0.112.2130 (Trial, Corporate) fix 09.03.2020
__________________________________________________

Для лечения использовать файл из папки FixFiles


!Обычно после применения лекарства служба лицензирования становится не нужна.
Поэтому по желанию ее можно как оставить, так и остановить, или удалить.

Файл "Del_Lic_Serv.bat" удаляет службу лицензирования (не обязателен к применению)


===================================================




P.S.
При отключении службы или удалении файла лицензии - информация не может быть получена, 
поэтому в заголовке окна и в about исчезает название установленной версии (Standard, Corporate, Enterprise).
